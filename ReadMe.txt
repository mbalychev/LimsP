Формирует текстовый файл в формате csv. 
Данные формируются в результате sql запросов к базе данных Limsprod. Данные это результат анализов по параметрам, которые находятся в файлах Units.txt и Formuls.txt. Простые результаты это результат одного анализа. Вычисляемые результаты — это данные нескольких или одного результата анализов, вычисляемые по формуле. Результаты выбираются самые свежие на 7.00 утра. Входе работы ведется учет ошибок, как при обращение к БД и получении результатов, так и в ходе вычисления и парсинга файлов.

Ход работы программы.
Получение результатов для простых результатов:
    • Парситься файл  Units.txt
    • Формируется запрос SQL к базе данных
    • Результат записывается в выходной файл
Получение результатов для вычисляемых результатов:
    • Парситься файл  Units.txt
    • Парситься файл  Formuls.txt
    • Формируется несколько запросов SQL к базе данных
    • Из результатов запросов по формуле (из файла Formuls) вычисляется значение
    • Результат записывается в выходной файл

Формат файла Units.txt, в локальной папке программы (при запуске с task scheduller c:\windows\system):
    • наименование установки
    • наименование  продукта
    • наименование точки отбора
    • порядковый номер показателя (назначается в процессе заполнения файла Units администратором)
    • наименование компонента анализа
    • все позиции с кодом больше 1000 не попдают в выходной файл (предназначены для вычислений)
Формат файла  Formuls.txt, в локальной папке программы (при запуске с task scheduller c:\windows\system):
    • порядковый номер показателя
    • формула для вычисления. Используются след.математические символы  +, -, *, /, (, ).
Формат выходного файла
    • код ЛИМС, типа «Lims_» + порядковый номер + «_3.txt»
    • результат, типа double
    • номер образца результат (для простого)
    • имя файла «\\kms-dfs01\s\MatBalans\Lims_» + дата получения результатов
Формат файла ошибок:
    • код ЛИМС, типа «Lims_» + порядковый номер
    • текст ошибки
    • результат полученный из бд (если есть)
    • имя файла, в локальной папке программы «Lims_Error» + дата получения результатов + «_3.txt»

База данных: limsprod
Framework:  EntityFramework
